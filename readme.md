
# Task Todo

Sistem ini mengatur alur task to do dengan mekanisme persetujuan (approval) berdasarkan tiga level role

## Backend

URL_API: [http://localhost:8080/](http://localhost:8080/)

### Cara Install

- Clone Repository

``` bash
git clone https://github.com/username/nama-repo.git
cd nama-repo
 ```

- Run Project
``` bash
go run main.go
 ```
